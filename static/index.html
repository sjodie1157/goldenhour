CapstoneBud API

Endpoints

App Endpoints
Path Method Description
/ GET Gets This Page
/register POST Registers a new user
/login POST Logs in a user
User/s Endpoints
Path Method Description
/users GET Gets all existing user accounts
/user/{id} GET Gets user by ID
/user/{id} PATCH Updates user by ID
/user/{id} DELETE DELETE user by ID
/users/search/{query} GET Search a user by name
/user/profile/me GET Gets current logged in user info
Post/s Endpoints
Path Method Description
/posts GET Gets all posts (user must be logged in)
/post/{postID} GET Gets a post by ID (user must be logged in)
/posts POST Add's a post (user must be logged in)
/post/{postID} PUT Add's a post (user must be logged in)
/post/{postID} DELETE Deletes a post (must be user that posted it)
Comment/s Endpoints
Path Method Description
/post/{postID}/comments GET Gets all comments (user must be logged in)
/post/{postID}/comment/{commentID} GET Gets a comment (user must be logged in)
/post/comment POST Add a comment to post (user must be logged in)
/post/{postID}/comment/{commentID} PUT Add a comment to post (user must be logged in)
/post/{postID}/comment/{commentID} DELETE Deletes a comment on post (user must be logged in)
/post/3/comments DELETE Delete all comments on post (user must be logged in)
